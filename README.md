# 🚀 FarmaStock API

Sistema backend desenvolvido em Java com Spring Boot para controle inteligente de estoque de farmácias pequenas.

---

## 🎯 Objetivo

Controlar estoque de medicamentos e sugerir reposição automática com base nas vendas.

---

## 🧠 Funcionalidades do MVP

- Cadastro de produtos
- Controle de estoque
- Registro de vendas
- Alerta de estoque mínimo
- Sugestão automática de pedido
- Sugestão de genéricos por princípio ativo

---

## 🛠 Tecnologias

- Java
- Spring Boot
- PostgreSQL
- Maven
